5. Create a `cleanCliOutput` helper that strips ANSI codes AND strips lines starting with `[plugins]` (OpenClaw warning lines that corrupt JSON):
```javascript
function cleanCliOutput(raw) {
  return stripAnsi(raw)
    .split('\n')
    .filter(line => !line.startsWith('[plugins]'))
    .join('\n')
    .trim();
}
```

6. Create the `dispatchAgent(taskId, agentId, message)` function:
   - Use a `db.transaction` that:
     a. Calls `stmts.updateTask.run(...)` to set the task's status to `'in_progress'` (read existing task first with `stmts.getTaskById.get(taskId)` to preserve other fields)
     b. Calls `stmts.insertRun.run({ task_id: taskId, agent_id: agentId, message })` and captures `lastInsertRowid` as `runId`
     c. Calls `stmts.insertActivity.run(...)` with event_type `'agent.dispatched'`, agent_id, task_id, summary `'Dispatched ' + agentId + ' for task #' + taskId`
     d. Returns `{ runId, updatedTask: stmts.getTaskById.get(taskId) }`
   - After the transaction, emit SSE events:
     - `bus.emit('agent:started', { run_id: runId, agent_id: agentId, task_id: taskId })`
     - `bus.emit('task:updated', updatedTask)`
     - `bus.emit('activity:new', { event_type: 'agent.dispatched', agent_id: agentId, task_id: taskId, summary: ... })`
   - Spawn the CLI using `execFile` (NEVER `exec` -- no shell, prevents injection):
     ```javascript
     const child = execFile('openclaw', [
       'agent', '--agent', agentId,
       '--message', message,
       '--json',
       '--timeout', '600'
     ], {
       timeout: 660000,       // 11 min hard kill (above CLI's 10 min)
       maxBuffer: 10 * 1024 * 1024,
       env: { ...process.env, PATH: process.env.PATH + ':/opt/homebrew/bin' }
     }, (error, stdout, stderr) => { ... });
     ```
   - Store in activeDispatches: `activeDispatches.set(runId, { process: child, agentId, taskId, startTime: Date.now() })`
   - In the execFile callback:
     - **On success (no error):** Clean stdout with `cleanCliOutput()`, try `JSON.parse`, extract `result.payloads?.map(p => p.text).join('\n')` as resultText (fallback to cleaned stdout if not JSON or no payloads). Calculate durationMs from startTime. Call `stmts.completeRun.run({ id: runId, status: 'completed', result_json: cleanedStdout, result_text: resultText, error: null, duration_ms: durationMs })`. Update task to 'review': read current task, call updateTask with status='review'. Insert activity. Emit `bus.emit('agent:completed', { run_id: runId, agent_id: agentId, task_id: taskId, duration_ms: durationMs, result_text: resultText ? resultText.substring(0, 200) : '' })`. Emit task:updated. Emit activity:new. Remove from activeDispatches.
     - **On error:** Determine status: `error.killed ? 'timeout' : 'failed'`. Calculate durationMs. Call `stmts.completeRun.run({ id: runId, status, result_json: null, result_text: null, error: error.message, duration_ms: durationMs })`. Do NOT change task status (leave in_progress for retry). Insert activity with appropriate summary. Emit `bus.emit('agent:failed', { run_id: runId, agent_id: agentId, task_id: taskId, error: error.message, duration_ms: durationMs })`. Emit activity:new. Remove from activeDispatches.
   - Return `runId` synchronously.

function formatElapsed(ms) {
  if (!ms || ms < 1000) return '0s';
  var s = Math.floor(ms / 1000);

  var m = Math.floor(s / 60);
